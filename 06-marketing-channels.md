## 1. Выбор стратегии и каналов

### 1.1 Стратегия в зависимости от этапа

| Этап | Цель | Приоритет каналов |
|------|------|-------------------|
| Запуск | Первые лиды, проверка спроса | Контекст (Google Ads), таргет (Meta), быстрый поток заявок |
| Рост | Стабильный поток, узнаваемость | Контекст + таргет + SEO + отзывы и рекомендации |
| Масштаб | Максимум заказов при приемлемом CAC | Масштабирование лучших каналов, ретаргетинг, агрегаторы/партнёры |

### 1.2 Критерии выбора канала

- **Где ищут клиенты:** поиск («детейлинг Берлин») → Google Ads и SEO; соцсети и визуал → Meta, локальные площадки.
- **Бюджет:** ограниченный старт — 1–2 канала с быстрой отдачей (часто контекст + один таргет); при росте — добавление SEO, агрегаторов.
- **Возможность измерить:** канал должен давать возможность проставлять источник в [CRM](03-crm.md) и считать CPL/CAC (см. [сквозная аналитика](07-analytics-and-tools.md)). Звонки учитывать через [коллтрекинг](07-analytics-and-tools.md).

### 1.3 Обзор каналов

| Канал | Тип | Что даёт | Когда подключать |
|-------|-----|----------|-------------------|
| **Сайт + SEO** | Органический трафик | Долгосрочные заявки по поиску | С [запуском сайта](05-website.md) |
| **Google Ads** | Платный трафик | Быстрые заявки по запросам («автополировка», «химчистка салона» и т.д.) | После [прайса](02-services-pricing.md) и посадочных страниц |
| **Meta (Facebook/Instagram)** | Таргетированная реклама | Охват, узнаваемость, заявки из соцсетей | Параллельно с сайтом или сразу после |
| **Рекомендации и отзывы** | Офлайн/онлайн | Низкая стоимость лида, доверие | С первых клиентов (Google-отзывы и т.п.) |
| **Локальные каталоги** (жёлтые страницы, отраслевые справочники) | Лиды/видимость | Дополнительные заявки и присутствие | После настройки [CRM](03-crm.md) |

## 2. Учёт источника в CRM

Для каждого канала в [CRM](03-crm.md) должна быть своя **источник (Source)**:

- **Стоимость лида (CPL)** = расходы на канал / количество лидов.
- Конверсия лид → заказ по каналу.
- **CAC (стоимость привлечения клиента)** = расходы на канал / количество полученных клиентов.
- Управление бюджетом: наращивать каналы с лучшим CAC/ROI. Без [сквозной аналитики и коллтрекинга](07-analytics-and-tools.md) часть лидов (звонки) будет теряться — источник будет «неизвестный».

## 3. Расчёт рекламных кампаний

### 3.1 Исходные данные (из вашего бизнеса)

| Показатель | Значение | Единица | Источник |
|------------|----------|--------|----------|
| **Средний чек** | € | за заказ | [Прайс](02-services-pricing.md), [CRM](03-crm.md) |
| **Валовая маржа на заказ** | % или € | | Себестоимость vs цена продажи |
| **Целевая конверсия лид → заказ** | % | | Оценка или факт (например 20–30%) |
| **Макс. допустимый CAC** | € | на клиента | см. ниже |
| **Месячный бюджет на рекламу** | € | | План из [постоянных расходов](02-services-pricing.md) или отдельно |

### 3.2 Максимально допустимый CAC (ориентир)

**CAC_макс ≈ Валовая маржа на заказ × допустимая доля на привлечение.**

Пример: маржа на заказ 150 €, готовы отдать на привлечение не более 30% → **CAC_макс ≈ 45 €**.  
При конверсии лид → клиент 25% получаем **CPL_макс ≈ 45 / 0,25 = 180 €** на лид.

### 3.3 Формулы

| Показатель | Формула |
|------------|---------|
| **CPL** (стоимость лида) | Бюджет на рекламу (€) / количество лидов |
| **CAC** (стоимость привлечения клиента) | Бюджет (€) / количество привлечённых клиентов = CPL / конверсия лид→клиент |
| **Целевое количество лидов** | Бюджет (€) / целевой CPL (€) |
| **Безубыточность** | CAC ≤ валовая маржа на клиента (или целевая маржа) |

### 3.4 Пример расчёта бюджета (EUR)

- Цель: **20 квалифицированных лидов в месяц**.
- Ориентир: CPL в Google Ads примерно **40–80 €**, в Meta **25–50 €** (зависит от региона и конкуренции).
- Допустим CPL **50 €** → бюджет **20 × 50 = 1 000 €/мес** на один канал.
- При конверсии 25% → 5 клиентов; при марже 200 € с клиента → 1 000 € маржи; при таком бюджете выходим в ноль, масштабирование — при лучшей конверсии или более высоком среднем чеке.

**Рекомендация:** сначала тестировать небольшие бюджеты (например 300–500 €/канал/мес), замерять CPL и конверсию в [CRM](03-crm.md), затем увеличивать бюджет по результатам.

**Зависимости от цены и наценки:** маржа на заказ задаёт верхнюю границу допустимого CAC и CPL; при изменении цен или [правил скидок](08-strategy-discounts-dependencies.md) лимиты и бюджет нужно пересчитывать. Полная цепочка и нормативы — в разделе [Стратегия, скидки и зависимости](08-strategy-discounts-dependencies.md).

## 4. Внедрение рекламных кампаний: пошагово

### Этап 1: Подготовка (до первых расходов на рекламу)

| Шаг | Действие | Ссылка |
|-----|----------|--------|
| 1.1 | Посадочные страницы с чёткими услугами и ценами (EUR) | [Сайт](05-website.md) |
| 1.2 | Форма/CTA с привязкой к **источнику** (UTM или отдельная URL под кампанию) | [CRM](03-crm.md), [Сайт](05-website.md) |
| 1.3 | В CRM создать источники «Google Ads», «Meta» и т.д. | [CRM](03-crm.md) |
| 1.4 | В аналитике настроить цели: отправка формы, клик по телефону; при необходимости — продуктовая аналитика (PostHog) для воронок и записей сессий | [Аналитика и коллтрекинг](07-analytics-and-tools.md), [PostHog](07-analytics-and-tools.md) |

### Этап 2: Тест кампаний (ограниченный бюджет)

| Шаг | Действие |
|-----|----------|
| 2.1 | **Google Ads:** поисковая кампания, 5–15 ключевых слов («автополировка [город]», «химчистка салона» и т.д.), бюджет например 15–25 €/день, объявления ведут на посадочную с ценами и формой заявки |
| 2.2 | **Meta:** таргетированные объявления (возраст, интересы авто/уход), картинка/видео + текст + ссылка на сайт, бюджет например 10–15 €/день |
| 2.3 | Длительность теста: не менее 2–3 недель. Все лиды вносить в CRM с правильным источником; звонки учитывать через [коллтрекинг](07-analytics-and-tools.md) |
| 2.4 | Анализ: CPL, конверсия лид → заказ, CAC по каналам |

### Этап 3: Оптимизация и масштабирование

| Шаг | Действие |
|-----|----------|
| 3.1 | Канал с лучшим CAC усиливать; слабый — сокращать или улучшать объявления/посадочные |
| 3.2 | Постепенно увеличивать бюджет (например +20–30% в неделю) при стабильных CPL и конверсии |
| 3.3 | A/B-тесты: тексты объявлений, картинки, варианты посадочных |
| 3.4 | Ежемесячный отчёт: бюджет (€), лиды, клиенты, CAC, выручка по кампаниям — на основе [CRM](03-crm.md) и [аналитики](07-analytics-and-tools.md) |

## 5. Минимальные метрики (в EUR где применимо)

- Количество лидов по источникам.
- Конверсия лид → заказ по каждому источнику (%).
- CPL и CAC по каналу.
- Средний чек по клиентам из кампаний.
- **ROI рекламы** = (Выручка по кампании − Расходы на кампанию) / Расходы на кампанию (если можно привязать выручку к источнику).

Все расчёты возможны только при корректном указании **источника** в [CRM](03-crm.md) для каждого лида (включая лиды из звонков при [коллтрекинге](07-analytics-and-tools.md)).

**Полный набор маркетинговых метрик:** когда считать, что нужно для расчёта, цели и ритм отчётности — в разделе [Метрики и отчётность](10-metrics-and-kpis.md).
