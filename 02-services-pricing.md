## 1. Как считать себестоимость: пошагово

### Шаг 1. Постоянные расходы (месяц, EUR)

Соберите все фиксированные затраты за месяц. Без них нельзя корректно распределить долю на одну услугу.


| Статья                                      | Сумма (€/мес) | Примечание                                |
| ------------------------------------------- | ------------- | ----------------------------------------- |
| Аренда                                      | 16000         | Зависит от локации                        |
| Коммуналка (электричество, вода, отопление) | 2000          |                                           |
| Амортизация оборудования                    | 4000          | Стоимость покупки / срок службы в месяцах |
| Зарплата руководства/офиса (доля)           | 8000          | Доля времени на администрирование         |
| Бюджет на маркетинг (план)                  | 0             | Месячный плановый расход                  |
| Страховка, бухгалтер, прочее                | 0             |                                           |
| **Итого постоянные расходы**                | **€30000**    |                                           |


**Рекомендация:** ведите отдельной строкой «Маркетинг», чтобы потом сравнивать с фактическими лидами и CAC (см. [каналы и расчёт кампаний](06-marketing-channels.md)).

- Пример: 1 мастер × 160 ч = 160 ч; 2 мастера × 160 ч = 320 ч.

Два подхода:

- **Б) Целевая ставка:** желаемая «нагрузка» часа после вычета материалов (например 35–50 €/ч).

**Ставка часа (€/ч):** 

### Шаг 4. Доля постоянных расходов на один час

**Постоянные расходы на час = Итого постоянные расходы / плановые часы.**

Пример: 4 000 € / 160 ч = **25 €/ч**. Каждый час работы должен «зарабатывать» 25 € на покрытие постоянных расходов.

### Шаг 5. Себестоимость одной услуги — формула

```
Себестоимость услуги (€) =
  Материалы (€)
  + (Время (ч) × Ставка часа (€/ч))
  + (Время (ч) × Постоянные расходы на час (€/ч))
```

Короче: объединяем ставку и долю постоянных в **полную стоимость часа**:

**Полная стоимость часа = Ставка часа + Постоянные расходы на час**

**Себестоимость услуги = Материалы + (Время × Полная стоимость часа)**

## 2. Карточка услуги

Для каждой услуги заведите карточку: так вы считаете себестоимость и задаёте нормы для [ERP](04-erp-inventory.md).

### 2.1 Шаблон карточки (на одну единицу услуги)


| Поле                         | Описание                             | Пример                    |
| ---------------------------- | ------------------------------------ | ------------------------- |
| **Услуга**                   | Название                             | Химчистка салона (полная) |
| **Единица**                  | Ед. измерения                        | 1 автомобиль              |
| **Время (ч)**                | Норма времени в часах                | 4                         |
| **Материалы**                | Список и расход на 1 ед.             | см. таблицу ниже          |
| **Стоимость материалов (€)** | Сумма на 1 услугу                    | 18,50                     |
| **Трудозатраты (€)**         | Время × Полная стоимость часа        | 4 × 60 = 240              |
| **Доля постоянных (€)**      | уже включена в полную стоимость часа | —                         |
| **Себестоимость (€)**        | Материалы + Трудозатраты             | 258,50                    |
| **Цена продажи (€)**         | Целевая цена (без НДС или с НДС)     | 399                       |
| **Маржа (%)**                | (Цена − Себестоимость) / Цена        | ок. 35%                   |


### 2.2 Пример: «Химчистка салона (полная)»

**Материалы (на 1 автомобиль):**


| Материал                      | Расход | Закупочная цена (€) | Стоимость (€) |
| ----------------------------- | ------ | ------------------- | ------------- |
| Средство для салона           | 0,5 л  | 12/л                | 6,00          |
| Микрофибра (шт.)              | 8      | 0,40                | 3,20          |
| Щётки/губки                   | —      | 1,50                | 1,50          |
| Прочее (ароматизаторы и т.д.) | —      | 2,00                | 2,00          |
| **Итого материалы**           |        |                     | **12,70**     |


**Время:** 4 ч  
**Полная стоимость часа:** 60 €/ч (включая долю постоянных)  
**Трудозатраты:** 4 × 60 = **240 €**  
**Себестоимость:** 12,70 + 240 = **252,70 €**  
**Цена продажи (без НДС):** например 389 € → **С НДС 19%:** 462,91 €  
**Маржа:** (389 − 252,70) / 389 ≈ **35%**

Эти же нормы (материалы на авто, время) используются в [ERP](04-erp-inventory.md) для списания и пересчёта при изменении закупочных цен. Детальная спецификация этой услуги с разбивкой по ролям и материалам (BOM + маршрут труда) — в [04-erp-inventory.md, раздел 4](04-erp-inventory.md).

## 3. Расширенный финансовый расчёт (для максимальной точности)

### 3.1 Точка безубыточности (в месяц, EUR)

**Точка безубыточности по выручке:** постоянные расходы / доля маржинальной прибыли в выручке.

**Маржинальная прибыль на единицу** = Цена продажи − Переменные затраты (материалы + ставка часа без постоянных).  
**Доля маржи** = Маржинальная прибыль на единицу / Цена продажи.

**Точка безубыточности (кол-во заказов)** = Постоянные расходы / Маржинальная прибыль на один заказ (средняя).

Пример: постоянные 4 000 €, средняя маржинальная прибыль на заказ 120 € → нужно **4 000 / 120 ≈ 34 заказа в месяц** для выхода в ноль.

### 3.2 Запас прочности

**Запас прочности (%)** = (Фактическая выручка − Выручка в точке безубыточности) / Фактическая выручка × 100. Показывает, насколько можно упасть по выручке до убытка.

### 3.3 Учёт сезонности и скидок

- Заложите в плановые часы и выручку сезонность (например лето/праздники).
- При акциях и скидках пересчитывайте минимальную цену: не ниже себестоимости + целевая доля постоянных.

### 3.4 Сезонность и календарь (когда что планировать)

Для детейлинга часто характерен рост спроса весной и осенью (подготовка к сезону); в отдельные месяцы возможен спад. Чтобы план был «до конца» по этапам года:

- **Календарь:** в какие месяцы закладывать больший [бюджет на рекламу](06-marketing-channels.md) и больше плановых заказов; когда запускать акции (например в низкий сезон для заполнения слотов); когда обновлять контент и креативы.
- **Связь с точкой безубыточности:** в низкий сезон выручка может падать — заранее заложить запас по деньгам или снизить переменные/постоянные где возможно; не нарушать [минимальную наценку](08-strategy-discounts-dependencies.md) при акциях.

## 5. Нормы расхода материалов (для ERP)

Для каждой услуги зафиксируйте **расход на одну единицу**. Тогда [ERP](04-erp-inventory.md) при закрытии заказа сможет списывать по нормам (автоматически или полуавтоматически).

Пример:


| Услуга           | Материал            | Расход на 1 ед. |
| ---------------- | ------------------- | --------------- |
| Химчистка салона | Средство для салона | 0,5 л           |
| Химчистка салона | Микрофибра          | 8 шт.           |
| Мойка кузова     | Шампунь             | 0,1 л           |
| Мойка кузова     | Микрофибра/перчатки | 3 шт.           |


При изменении закупочных цен пересчитывайте **себестоимость** услуг и при необходимости корректируйте цену продажи. Себестоимость и маржа — основа для [лимитов на рекламу (CAC/CPL)](08-strategy-discounts-dependencies.md) и для [минимальной цены при скидках](08-strategy-discounts-dependencies.md).

## 6. Пакеты и акции

- **Пакеты** (например «Мойка + химчистка салона»): себестоимость и цена как сумма услуг минус скидка; фиксируйте скидку в % или €.
- **Акции:** ограничивайте по срокам; проверяйте, что цена не ниже себестоимости (с учётом доли постоянных). Правила и этап дисконтирования (когда какие скидки допустимы, минимальная наценка) — в плане [Стратегия, скидки и зависимости](08-strategy-discounts-dependencies.md).
