# ERP и склад (EUR)

> [← Главный план](main.md)

Справочники материалов и **ролей**, нормы труда по специалистам, детальные спецификации (BOM + маршрут труда) на услуги и списание по заказам.

---

## 1. Задачи учёта

- Контроль остатков расходников и запчастей.
- **Себестоимость заказа:** материалы (по нормам) + труд по ролям (часы × ставка).
- Планирование закупок (что и когда докупить, мин. остатки).
- Отчётность: выручка, себестоимость, маржа по услугам, по периоду и при необходимости по ролям/центрам затрат.

## 2. Справочники (номенклатура и роли)

| Поле                | Назначение                                | Пример                              |
| ------------------- | ----------------------------------------- | ----------------------------------- |

### 2.2 Справочник ролей (труд)

Чтобы считать себестоимость по разным специалистам и планировать загрузку, вводятся **роли** с полной стоимостью часа (зарплата + налоги + доля постоянных расходов). Нормы на услугу задаются в часах по каждой роли.


| Поле                        | Назначение                                  | Пример             |
| --------------------------- | ------------------------------------------- | ------------------ |
| Код роли                    | Уникальный идентификатор                    | R-MASTER, R-JUNIOR |
| Наименование                | Мастер, помощник, специалист по коже и т.д. | Мастер детейлинга  |
| Полная стоимость часа (€/ч) | Ставка для расчёта себестоимости            | 62,00              |
| Примечание                  | Зарплатная вилка, квалификация              | Опыт от 2 лет      |


**Пример ролей для детейлинга:**


| Код        | Роль                                             | €/ч (пример) |
| ---------- | ------------------------------------------------ | ------------ |
| R-MASTER   | Мастер детейлинга (кузов, полировка, защита)     | 62           |
| R-INTERIOR | Специалист по салону (химчистка, кожа, текстиль) | 58           |
| R-JUNIOR   | Помощник / мойщик (мойка, подсобные работы)      | 42           |


Расчёт полной стоимости часа по ролям — в [Услуги и прайс](02-services-pricing.md); при разных окладах и нагрузке ставки по ролям различаются. Роли в ERP должны соответствовать реальным [должностям и должностным инструкциям](13-job-descriptions.md) (мастер, специалист по салону, помощник и т.д.).

## 3. Нормы на услугу: материалы и труд (BOM + маршрут)

Для каждой [услуги из прайса](02-services-pricing.md) в ERP задаются две вещи:

1. **Нормы материалов (BOM)** — что и сколько списывается на одну единицу услуги.
2. **Нормы труда (маршрут)** — сколько часов по каждой роли закладывается на одну единицу услуги.

Так можно считать себестоимость автоматически и списывать материалы при закрытии заказа в [CRM](03-crm.md).

## 5. Движения по складу

- **Приход** — закупка: дата, поставщик, документ, позиции (код, название, кол-во, цена в €). Обновление остатков и при необходимости средняя цена.
- **Расход** — по заказу: привязка к сделке/заказу из [CRM](03-crm.md), списание по нормам BOM указанной услуги (или вручную при отклонениях). Опционально: фиксация фактических часов по ролям для анализа.
- **Остаток** — текущий остаток по каждой позиции; предупреждение при достижении минимального остатка.

## 6. Этап внедрения и порядок шагов

### 6.1 Шаги внедрения (последовательность)


| №   | Шаг                    | Результат                                                                                                                                                                             |
| --- | ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1   | Справочник материалов  | Каталог с кодом, названием, ед. изм., ценой, мин. остатком, поставщиком ([раздел 2.1](#21-справочник-материалов)).                                                                    |
| 2   | Справочник ролей       | Роли (мастер, специалист по салону, помощник) с полной стоимостью часа; согласование с [должностными инструкциями](13-job-descriptions.md) ([раздел 2.2](#22-справочник-ролей-труд)). |
| 3   | BOM для первых услуг   | Нормы расхода материалов на 1–2 ключевые услуги по [карточкам](02-services-pricing.md); таблица как в [разделе 3.1](#31-нормы-материалов-bom---пример-фрагмента).   |
| 4   | Маршруты труда         | Часы по ролям на те же услуги ([раздел 3.2](#32-нормы-труда-маршрут---пример)).                                                                                                       |
| 5   | Учёт прихода и расхода | Регистрация закупок (приход), списание по заказам по нормам BOM (расход), остатки; на старте — таблицы (Sheets/Excel) или простой модуль.                                             |
| 6   | Связка с CRM           | При переводе сделки в «Выполнено» в CRM — списание материалов по коду услуги и количеству; автоматически или полуавтоматически.                                                       |
| 7   | Расширение и масштаб   | BOM и маршрут для всех услуг; планирование закупок по мин. остаткам; отчётность по себестоимости и по ролям.                                                                          |


### 6.2 Уровни зрелости (Старт / Рост / Масштаб)


| Этап    | Что делать                                                                                                                                                    |
| ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Старт   | Справочник материалов и ролей в таблицах (Sheets/Excel); одна-две услуги с полной спецификацией (BOM + маршрут), как в разделе 4. Ручное списание по заказам. |
| Рост    | Модуль склада/закупок в той же системе, что и CRM, или отдельный ERP-модуль: приходы, расходы по заказу из CRM, остатки, отчёты по себестоимости.             |
| Масштаб | Все услуги с BOM и маршрутом; планирование закупок по мин. остаткам и плану заказов; отчётность по ролям и центрам затрат.                                    |


Перечень [программ и инструментов](07-analytics-and-tools.md) — в разделе по аналитике и ПО.

## 7. Связь с другими разделами

- Нормы материалов и труд по ролям согласованы с [карточками услуг и калькуляцией](02-services-pricing.md); при упрощённой модели «один усреднённый час» в 02 можно задать одну роль с суммарными часами.
- Заказ для списания приходит из [CRM](03-crm.md) после перевода сделки в статус «Выполнено»; при интеграции ERP–CRM передаётся код услуги и количество.
- Лимиты на скидки и рекламу (CAC, CPL) зависят от маржи и себестоимости — [08](08-strategy-discounts-dependencies.md).
- **Бухгалтерия:** себестоимость заказа (материалы + труд по нормам) может передаваться для учёта затрат и расчёта маржи; выручка и счета (Rechnung) ведутся в бухгалтерии на основе данных из CRM. Полная связка CRM–ERP–бухгалтерия — в [15](15-business-process-integration.md).
