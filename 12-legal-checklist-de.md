# Чек-лист: юрлицо и регламенты

|-------|--------|------------|
| Регистрация бизнеса (Gewerbe и т.п.) в соответствии с местным правом | ☐ | |
| Решение по НДС (USt.): регистрация как плательщик при необходимости; применение ставки 19% к услугам | ☐ | |
